# SocialMediaApp2 - Backend

## Overview
This is the backend for **SocialMediaApp2**, a social media application providing authentication, user management, and token generation functionalities.

## Technologies Used
- Node.js
- Express.js
- MongoDB (via Mongoose)
- JSON Web Tokens (JWT)
- dotenv
